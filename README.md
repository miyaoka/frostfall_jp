Frostfall - Hypothermia Camping Survival 私家版日本語化
============

Skyrimのmod「Frostfall - Hypothermia Camping Survival」の日本語化、のさらに私家版調整です。  
日本語版ファイルを元に調整を加えています。

### 主な調整点
* エフェクト名に接頭辞を追加
* 体温変動時の通知メッセージを調整	

### 利用について
特に無いです。勝手に使ってもらって結構です。
